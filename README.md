# Projects Collection

A collection of Python utility and game applications showcasing basic programming concepts.

## 📁 Projects Overview

### 1. **File Manager** 📂
A command-line utility for file operations.
- Create, read, update, and delete files
- View all files in a directory
- Simple and intuitive interface
- **Location**: `filemanger/`
- **Main File**: `File manger.py`
- [View Details](filemanger/README.md)

### 2. **Rent Calculator** 💰
A financial calculator for splitting shared expenses.
- Calculate individual shares of rent and utilities
- Support for multiple residents
- Multiple expense categories (rent, food, electricity, water, internet)
- Input validation and currency formatting
- Save results to file
- **Location**: `Rent Calculator/`
- **Main File**: `app.py`
- [View Details](Rent%20Calculator/README.md)

### 3. **Rock Paper Scissor Game** 🎮
A classic game implementation where players compete against the computer.
- Interactive gameplay
- Score tracking
- Random computer moves
- Multiple rounds with running scores
- **Location**: `Rock Paper Scissor/`
- **Main File**: `main.py`
- [View Details](Rock%20Paper%20Scissor/README.md)

### 4. **Tic Tac Toe Game** 🎮
A two-player Tic Tac Toe game with graphical interface.
- Two-player gameplay
- Win and draw detection
- Visual winning highlights
- Reset button for multiple games
- Player turn indicator
- **Location**: `Tic Tac Toe/`
- **Main File**: `main.py`
- [View Details](Tic%20Tac%20Toe/README.md)

### 5. **Text Editor** 📝
A simple yet functional text editor with file operations.
- Create, open, and save files
- Clean menu-based interface
- Text wrapping for readability
- Cross-platform compatible
- **Location**: `Text editior/`
- **Main File**: `main.py`
- [View Details](Text%20editior/README.md)

### 6. **Digital Clock** ⏰
A real-time digital clock with date display.
- Live time updates every second
- 12-hour format with AM/PM
- Current date display
- Large, readable font
- Minimal design
- **Location**: `Digital Clock/`
- **Main File**: `main.py`
- [View Details](Digital%20Clock/README.md)

## 🔧 Project Details
- **Tic Tac Toe**: GUI development (Tkinter), game logic, win condition detection, event handling

Each project folder contains:
- **Main Python file**: The executable script

- **Sample files**: Example or test data files
